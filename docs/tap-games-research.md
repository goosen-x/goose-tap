## Механики и геймплей

### Базовые механики

```
┌─────────────────────────────────────────────────┐
│                 CORE LOOP                        │
├─────────────────────────────────────────────────┤
│                                                  │
│   TAP ──→ EARN COINS ──→ UPGRADE ──→ EARN MORE  │
│    ↑                                      │      │
│    └──────────────────────────────────────┘      │
│                                                  │
└─────────────────────────────────────────────────┘
```

### Типы механик

| Механика | Описание | Примеры |
|----------|----------|---------|
| **Pure Tap** | Простой клик по экрану | Notcoin |
| **Swipe** | Свайп вместо тапа | Yescoin |
| **Combo Tap** | Комбинации нажатий | Hamster Kombat |
| **Merge** | Объединение элементов | Catizen |
| **Idle** | Пассивный доход | Большинство игр |
| **PvP/PvE** | Бои с игроками/ботами | MemeFi |

### Системы прогрессии

1. **Уровни персонажа** — опыт, прокачка
2. **Апгрейды** — увеличение дохода за тап/час
3. **Карточки** — коллекционные элементы с бонусами
4. **Лиги** — соревнование с другими игроками
5. **Кланы** — групповые активности
6. **Задания** — ежедневные/еженедельные цели

### Монетизация (до токена)

- Реферальная программа
- Премиум подписка
- Буст за просмотр рекламы
- TON Stars / внутриигровые покупки

```
└────────────────────────────────────────┘
```

| Проект | Игрокам | Команде | Ликвидность |

- **Airdrop**: обычно 10-50% сразу, остальное через 10-12 месяцев
- **Команда**: lock 6-12 месяцев, vesting 24-36 месяцев
- **Листинг**: основные биржи (Binance, OKX, Bybit)

## Проблемы и критика

### Основные проблемы тапалок

| Проблема | Описание | % жалоб |
|----------|----------|---------|
| **Трудности вывода** | Сложность получения токенов | 35.9% |
| **Падение цены** | Дамп после листинга | — |
| **Разочарование аирдропом** | Маленькие награды | — |
| **Однообразный геймплей** | Скука от тапания | — |
| **Отсутствие utility** | Токен без применения | — |

### Критика модели

> "Tap-to-earn игры привлекают миллионы пользователей, но большинство не обеспечивают долгосрочной ценности. После аирдропа 80% пользователей уходят."
> — BeInCrypto

### Уроки провалов

1. **Hamster Kombat** — падение на 40% после листинга, массовое недовольство аирдропом
2. **Play-to-Earn** (Axie Infinity) — схлопывание экономики, урок для tap-to-earn
3. **Краткосрочный хайп** — Google Trends показывает падение интереса на 80%

## Рекомендации

### Что делать

1. **Фокус на геймплее** — интересные механики важнее токеномики
2. **Быстрый старт** — минимум барьеров для входа
3. **Социальные механики** — кланы, лиги, рефералы
4. **Постоянные обновления** — новый контент каждую неделю

### Чего избегать

1. **Пустых обещаний** — не гарантировать заработок
2. **Сложной токеномики** — простое понятное распределение
3. **Долгого ожидания** — листинг в разумные сроки
4. **Pay-to-win** — баланс между бесплатными и платными игроками
